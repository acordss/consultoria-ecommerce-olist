# 🚀 Consultoría de Datos: Optimización de E-commerce Olist

Este proyecto resuelve un problema crítico: **¿Por qué los clientes no vuelven a comprar?** 
### 📈 Impacto del Proyecto
Mediante el análisis de 100k órdenes de venta, he identificado:
* **Tasa de Churn:** Clientes que han abandonado la plataforma.
* **Segmentación VIP:** Identificación del 20% de clientes que generan el 80% del ingreso.
* **Mapa de Calor:** Regiones con mayor ticket promedio pero baja retención.

### 🛠️ Herramientas Utilizadas
* **Python (Pandas):** Limpieza y unión de 3 bases de datos relacionales.
* **Plotly:** Visualización interactiva de la salud del negocio.
* **Ingeniería de Sistemas:** Estructura de datos optimizada para análisis rápido.
